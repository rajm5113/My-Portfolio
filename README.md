## 💻 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or bun

### Installation
1. **Clone the repository**:
   ```bash
   git clone https://github.com/rajm5113/My-Portfolio.git
   ```
2. **Navigate to the directory**:
   ```bash
   cd My-Portfolio
   ```
3. **Install dependencies**:
   ```bash
   npm install
   ```
4. **Run development server**:
   ```bash
   npm run dev
   ```
